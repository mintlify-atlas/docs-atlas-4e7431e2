The `<Spacer>` component is a flexible space that expands along the major axis of its containing layout. It's a convenient shortcut for creating space between elements.

## Basic Usage

```tsx
import {render, Box, Text, Spacer} from 'ink';

const App = () => (
  <Box>
    <Text>Left</Text>
    <Spacer />
    <Text>Right</Text>
  </Box>
);

render(<App />);
```

Output:
```
Left                                                                      Right
```

## How It Works

`<Spacer>` is implemented as:

```tsx
<Box flexGrow={1} />
```

It expands to fill all available space in the flex container's main axis direction.

## Props

`<Spacer>` accepts no props. It's a simple component that just fills space.

## Examples

### Horizontal Spacing (Row Layout)

In the default row layout, `<Spacer>` pushes items apart horizontally:

```tsx
import {Box, Text, Spacer} from 'ink';

const Header = () => (
  <Box>
    <Text bold>My App</Text>
    <Spacer />
    <Text dimColor>v1.0.0</Text>
  </Box>
);
```

Output:
```
My App                                                                  v1.0.0
```

### Vertical Spacing (Column Layout)

In a column layout, `<Spacer>` pushes items apart vertically:

```tsx
import {Box, Text, Spacer} from 'ink';

const App = () => (
  <Box flexDirection="column" height={10}>
    <Text>Top</Text>
    <Spacer />
    <Text>Bottom</Text>
  </Box>
);
```

Output:
```
Top







Bottom
```

### Multiple Spacers

Multiple spacers divide space equally:

```tsx
import {Box, Text, Spacer} from 'ink';

const App = () => (
  <Box>
    <Text>Left</Text>
    <Spacer />
    <Text>Center</Text>
    <Spacer />
    <Text>Right</Text>
  </Box>
);
```

Output:
```
Left                          Center                          Right
```


## Spacer vs. Other Spacing Methods

### Spacer

Best for: Dynamic, flexible spacing that adapts to container size

```tsx
<Box>
  <Text>Left</Text>
  <Spacer />
  <Text>Right</Text>
</Box>
```

### Fixed Margin

Best for: Consistent, predictable spacing

```tsx
<Box>
  <Text>Left</Text>
  <Text marginLeft={10}>Right</Text>
</Box>
```

### justifyContent

Best for: Specific alignment patterns

```tsx
<Box justifyContent="space-between">
  <Text>Left</Text>
  <Text>Right</Text>
</Box>
```

### flexGrow

Best for: Custom flexible spacing with specific ratios

```tsx
<Box>
  <Text>Left</Text>
  <Box flexGrow={2} /> {/* Takes 2x space */}
  <Text>Middle</Text>
  <Box flexGrow={1} /> {/* Takes 1x space */}
  <Text>Right</Text>
</Box>
```

## Common Patterns

### Push to Edges

```tsx
// Push one item to the right

  <Text>Content</Text>
  <Spacer />
  <Text>→</Text>
</Box>

// Push one item to the left
<Box>
  <Text>←</Text>
  <Spacer />
  <Text>Content</Text>
</Box>
```

## Implementation Note

If you need more control over spacing, you can create custom spacer components:

```tsx
// Custom spacer with specific flex ratio
const CustomSpacer = ({flex = 1}) => <Box flexGrow={flex} />;

// Usage
<Box>
  <Text>Left</Text>
  <CustomSpacer flex={2} />  {/* 2x the space */}
  <Text>Middle</Text>
  <CustomSpacer flex={1} />  {/* 1x the space */}
  <Text>Right</Text>
</Box>
```
