The Chat App example demonstrates how to build an interactive CLI application that handles user input, manages a list of messages, and updates the UI in real-time.

## What This Example Demonstrates

- Using the `useInput` hook to handle keyboard input
- Managing dynamic lists with state
- Building multi-line layouts with `<Box>`
- Handling special keys (Enter, Backspace, Delete)
- Creating interactive forms in the terminal

## The Code

```tsx
import React, {useState} from 'react';
import {render, Text, Box, useInput} from 'ink';

let messageId = 0;

function ChatApp() {
  const [input, setInput] = useState('');

  const [messages, setMessages] = useState<
    Array<{
      id: number;
      text: string;
    }>
  >([]);

  useInput((character, key) => {
    if (key.return) {
      if (input) {
        setMessages(previousMessages => [
          ...previousMessages,
          {
            id: messageId++,
            text: `User: ${input}`,
          },
        ]);
        setInput('');
      }
    } else if (key.backspace || key.delete) {
      setInput(currentInput => currentInput.slice(0, -1));
    } else {
      setInput(currentInput => currentInput + character);
    }
  });

  return (
    <Box flexDirection="column" padding={1}>
      <Box flexDirection="column">
        {messages.map(message => (
          <Text key={message.id}>{message.text}</Text>
        ))}
      </Box>

      <Box marginTop={1}>
        <Text>Enter your message: {input}</Text>
      </Box>
    </Box>
  );
}

render(<ChatApp />);
```

## Running the Example

```bash
npm run example examples/chat
```

Type a message and press **Enter** to add it to the chat. Use **Backspace** or **Delete** to edit your input.

## How It Works

### State Management

```tsx
const [input, setInput] = useState('');
const [messages, setMessages] = useState<Array<{id: number; text: string}>>([]);
```

We maintain two pieces of state:
- `input` - The current text being typed
- `messages` - An array of all sent messages

### Input Handling

```tsx
useInput((character, key) => {
  if (key.return) {
    // Handle Enter key
  } else if (key.backspace || key.delete) {
    // Handle Backspace/Delete
  } else {
    // Handle regular character input
  }
});
```

The `useInput` hook provides a callback that fires for every keypress. It receives:
- `character` - The typed character (empty string for special keys)
- `key` - An object with boolean flags for special keys

### Sending Messages

```tsx
if (key.return) {
  if (input) {
    setMessages(previousMessages => [
      ...previousMessages,
      {
        id: messageId++,
        text: `User: ${input}`,
      },
    ]);
    setInput('');
  }
}
```

When Enter is pressed:
1. Check if there's input to send
2. Add a new message to the messages array
3. Clear the input field

### Deleting Characters

```tsx
else if (key.backspace || key.delete) {
  setInput(currentInput => currentInput.slice(0, -1));
}
```

Backspace and Delete remove the last character using `slice(0, -1)`.

### Adding Characters

```tsx
else {
  setInput(currentInput => currentInput + character);
}
```

Regular keystrokes append the character to the input.

### Layout

```tsx
<Box flexDirection="column" padding={1}>
  <Box flexDirection="column">
    {messages.map(message => (
      <Text key={message.id}>{message.text}</Text>
    ))}
  </Box>

  <Box marginTop={1}>
    <Text>Enter your message: {input}</Text>
  </Box>
</Box>
```

The layout uses nested `<Box>` components:
- Outer box: Column layout with padding
- Messages box: Displays all messages vertically
- Input box: Shows the current input with a prompt

## Key Concepts

### The useInput Hook

The `useInput` hook is essential for interactive CLIs. Key properties available:

```tsx
useInput((input, key) => {
  // Special keys
  key.return      // Enter
  key.escape      // Escape
  key.upArrow     // Up arrow
  key.downArrow   // Down arrow
  key.leftArrow   // Left arrow
  key.rightArrow  // Right arrow
  key.backspace   // Backspace
  key.delete      // Delete
  key.tab         // Tab
  key.shift       // Shift modifier
  key.ctrl        // Ctrl modifier
  key.meta        // Meta/Alt modifier
});
```

### List Management

Always use unique keys when rendering lists:

```tsx
{messages.map(message => (
  <Text key={message.id}>{message.text}</Text>
))}
```

The `key` prop helps React efficiently update the list.

## Enhancements

### Add Message Timestamps

```tsx
if (key.return) {
  if (input) {
    setMessages(previousMessages => [
      ...previousMessages,
      {
        id: messageId++,
        text: `User: ${input}`,
        timestamp: new Date(),
      },
    ]);
    setInput('');
  }
}

// In render:
{messages.map(message => (
  <Box key={message.id}>
    <Text dimColor>{message.timestamp.toLocaleTimeString()}</Text>
    <Text> - {message.text}</Text>
  </Box>
))}
```

### Limit Message History

```tsx
setMessages(previousMessages => [
  ...previousMessages.slice(-50), // Keep last 50 messages
  {
    id: messageId++,
    text: `User: ${input}`,
  },
]);
```

### Add Different Message Types

```tsx
type Message = {
  id: number;
  text: string;
  type: 'user' | 'system' | 'error';
};

// In render:
{messages.map(message => (
  <Text 
    key={message.id}
    color={
      message.type === 'error' ? 'red' :
      message.type === 'system' ? 'yellow' :
      'white'
    }
  >
    {message.text}
  </Text>
))}
```

### Add Cursor Position

```tsx
const [cursorPos, setCursorPos] = useState(0);

useInput((character, key) => {
  if (key.leftArrow) {
    setCursorPos(pos => Math.max(0, pos - 1));
  } else if (key.rightArrow) {
    setCursorPos(pos => Math.min(input.length, pos + 1));
  }
  // ... handle text insertion at cursor position
});
```
