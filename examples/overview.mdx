The Ink examples demonstrate real-world usage patterns and best practices for building beautiful CLI applications. Each example is a complete, working application that you can run and learn from.

## Running Examples

All examples are located in the `examples` directory of the Ink repository. You can run any example using:

```bash
npm run example examples/[example-name]
```

For instance:

```bash
npm run example examples/counter
npm run example examples/borders
npm run example examples/chat
```

## Example Categories

### Basic Examples

These examples demonstrate fundamental Ink concepts:

- **[Counter](/examples/counter)** - A simple counter using state and effects
- **[Chat App](/examples/chat-app)** - Interactive input handling and state management
- **[Select Input](/examples/select-input)** - Building an interactive list selector

### Styling & Layout

Learn how to style and layout your CLI components:

- **[Borders and Styling](/examples/borders-and-styling)** - Using border styles and colors
- **[Layout Patterns](/examples/layout-patterns)** - Common flexbox layout patterns

### Component Libraries

Discover the ecosystem of reusable components and hooks:

- **[Useful Components](/examples/useful-components)** - Third-party components for Ink
- **[Useful Hooks](/examples/useful-hooks)** - Custom hooks for common patterns

## What You'll Learn

Through these examples, you'll learn:

- **State Management** - How to use React hooks with Ink
- **User Input** - Handling keyboard input and user interactions
- **Layout** - Building responsive layouts with Flexbox
- **Styling** - Applying colors, borders, and text styles
- **Components** - Creating reusable, composable components
- **Patterns** - Common patterns for CLI applications
