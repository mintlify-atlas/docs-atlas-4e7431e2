Get up and running with Ink in just a few minutes. This guide will walk you through creating your first interactive CLI app.

## Create Your First App

The fastest way to get started is using `create-ink-app`:

<Steps>
  <Step title="Create a new project">
    Run the following command to scaffold a new Ink app:

    <CodeGroup>
    ```bash npm
    npx create-ink-app my-ink-cli
    ```

    ```bash yarn
    yarn create ink-app my-ink-cli
    ```

    ```bash pnpm
    pnpm create ink-app my-ink-cli
    ```
    </CodeGroup>

    For TypeScript support, add the `--typescript` flag:

    ```bash
    npx create-ink-app --typescript my-ink-cli
    ```
  </Step>

  <Step title="Navigate to your project">
    ```bash
    cd my-ink-cli
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Run your app">
    ```bash
    npm start
    ```

    You should see "Hello, Stranger" output in your terminal.
  </Step>
</Steps>

## Build a Simple Counter

Let's build a counter that increments every 100ms. Create a file called `counter.tsx`:

```tsx counter.tsx
import React, {useState, useEffect} from 'react';
import {render, Text} from 'ink';

const Counter = () => {
  const [counter, setCounter] = useState(0);

  useEffect(() => {
    const timer = setInterval(() => {
      setCounter(previousCounter => previousCounter + 1);
    }, 100);

    return () => {
      clearInterval(timer);
    };
  }, []);

  return <Text color="green">{counter} tests passed</Text>;
};

render(<Counter />);
```

Run it with:

```bash
npx tsx counter.tsx
```

### How It Works

<Accordion title="Component Structure">
  The `Counter` component is a regular React functional component. It uses React hooks like `useState` and `useEffect` just as you would in a browser-based React app.
</Accordion>

<Accordion title="State Management">
  `useState(0)` initializes the counter to 0. The `setCounter` function updates the state, triggering a re-render of the component.
</Accordion>

<Accordion title="Side Effects">
  `useEffect` sets up an interval that increments the counter every 100ms. The cleanup function (returned from `useEffect`) clears the interval when the component unmounts.
</Accordion>

<Accordion title="Rendering">
  The `render()` function from Ink renders your React component to the terminal instead of the DOM. The `Text` component displays colored text.
</Accordion>

## Add User Input

Let's make it interactive by adding keyboard controls:

```tsx interactive.tsx
import React, {useState} from 'react';
import {render, Text, Box, useInput} from 'ink';

const directions = {
  ArrowUp: '↑',
  ArrowDown: '↓',
  ArrowLeft: '←',
  ArrowRight: '→',
};

const Robot = () => {
  const [x, setX] = useState(1);
  const [y, setY] = useState(1);

  useInput((input, key) => {
    if (key.leftArrow) {
      setX(Math.max(1, x - 1));
    }

    if (key.rightArrow) {
      setX(Math.min(20, x + 1));
    }

    if (key.upArrow) {
      setY(Math.max(1, y - 1));
    }

    if (key.downArrow) {
      setY(Math.min(10, y + 1));
    }

    if (input === 'q') {
      process.exit(0);
    }
  });

  return (
    <Box flexDirection="column">
      <Text>
        Use arrow keys to move the robot. Press 'q' to exit.
      </Text>
      <Box height={12} borderStyle="single">
        <Box marginLeft={x} marginTop={y}>
          <Text>🤖</Text>
        </Box>
      </Box>
    </Box>
  );
};

render(<Robot />);
```

Run it with arrow keys to control the robot:

```bash
npx tsx interactive.tsx
```

## Understanding the Code

### Components

Ink provides several built-in components:

- **`<Text>`** - Display text with colors and styling
- **`<Box>`** - Layout container with Flexbox properties
- **`<Newline>`** - Insert line breaks
- **`<Spacer>`** - Flexible spacing

### Hooks

Ink provides React hooks for terminal interactions:

- **`useInput()`** - Handle keyboard input
- **`useApp()`** - Control app lifecycle (exit, etc.)
- **`useStdin()`** - Access stdin stream
- **`useStdout()`** - Access stdout stream
- **`useFocus()`** - Manage component focus
